# dot-style

ドット絵風地図スタイル (Pixel art map style)

レトロゲーム風の鮮やかな配色で、ドット絵ゲームの世界を連想させる地図スタイルです。

## プレビュー

https://geoloniamaps.github.io/dot-style/

## 使い方

```
https://geoloniamaps.github.io/dot-style/style.json
```

## 開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`docs/style.json` が生成されます。GitHub Pages で配信されます。
